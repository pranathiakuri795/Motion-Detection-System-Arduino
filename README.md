# Motion Detection System Using Arduino UNO and PIR Sensor

## Description
This project demonstrates a Motion Detection System using Arduino UNO and a PIR motion sensor in the Wokwi simulator. The PIR sensor detects human movement and sends signals to the Arduino UNO. When motion is detected, the LED turns ON and a message is displayed on the Serial Monitor. If there is no motion, the LED remains OFF. This project helps in understanding sensor interfacing, Arduino programming, and basic automation concepts in embedded systems.

## Components Used
- Arduino UNO
- PIR Motion Sensor
- LED
- Resistor
- Jumper Wires

## Software Used
- Wokwi Simulator
- Arduino IDE

## Applications
- Security Systems
- Smart Home Automation
- Automatic Lighting Systems
- Motion Detection Alarms
